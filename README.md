## 작품 배경
1. 아이들이 부모님이 집에 안계시거나 바빠서 책을 읽어줄 수 없는 상황일 때 혼자 책을 읽을 수 있는 방법이 어떤것이 있을까 고민
2. 누군가가 동화책을 읽어주게되면 아이들도 집중해서 들을수 있을 것이라고 생각
3. 부모님의 목소리를 녹음해서 동화책에서 소리내어 읽어 줄 수 있는 앱을 개발하기
